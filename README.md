Project Overview: Mr. Budget (Financial Control Dashboard)
Mr. Budget is a modern, responsive fintech web application designed to give users absolute control over their personal finances. Built with a user-centric approach, the platform provides real-time financial tracking, budget planning, and expense management in a single, streamlined interface.

Technical Stack
Frontend UI/UX: Designed using Stitch for a cohesive, component-driven design system and clean typography.

Frontend Framework: Built with React.js, ensuring a highly responsive, single-page application (SPA) experience with smooth state management.

Backend & Database: Powered by Firebase for real-time data synchronization, secure user authentication, and scalable cloud storage.

Key Features & UI Walkthrough
The dashboard is structured to give users an immediate, clear snapshot of their financial health:

Total Balance Card: Displays the user's aggregated wealth across all linked accounts, complete with a dynamic percentage indicator showing month-over-month growth or trends.

Icon-Enhanced Budget Status: Features visual, interactive progress bars paired with intuitive category icons (e.g., a house for Rent & Housing, a shopping cart for Groceries, and popcorn/film for Entertainment). This addition allows users to instantly identify and track their actual spending against set limits.

Upcoming Bills Planner: A proactive financial calendar widget that highlights upcoming due dates (e.g., automated bill reminders with "Settle" actions) to prevent late fees and improve cash flow management.

Recent Transactions Ledger: A dynamic, real-time data table showing recent financial activity. The ledger now features corresponding visual icons for every transaction type (such as a grocery bag or a power plug for utilities), allowing for rapid scannability by category, date, and amount.

Global Actions: A persistent floating action button (+ Add Transaction) and intuitive sidebar navigation make adding income or expenses seamless from any view.

Design & UX Highlights
Enhanced Visual Recognition: The integration of tailored icons reduces cognitive load, allowing users to categorize their spending habits in a split second without reading the text.

Clean Visual Hierarchy: Uses a balanced color palette—neutral backgrounds paired with vibrant green accents for positive metrics and primary actions—to maintain UI clarity.

Multi-Currency Support: Features native localization, including localized currency displays (such as Pakistani Rupee - PKR) for targeted regional tracking.

Accessibility & Scannability: Large metric callouts, progress trackers, and graphic iconography ensure users can assess their budget status in under three seconds.
