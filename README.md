Mr. Budget is a modern, responsive fintech web application designed to give users absolute control over their personal finances. Built with a user-centric approach, the platform provides real-time financial tracking, budget planning, and expense management in a single, streamlined interface.

🛠️ Technical Stack
🎨 Frontend UI/UX: Designed using Stitch for a cohesive, component-driven design system and clean typography.

⚛️ Frontend Framework: Built with React.js, ensuring a highly responsive, single-page application (SPA) experience with smooth state management.

🔥 Backend & Database: Powered by Firebase for real-time data synchronization, secure user authentication, and scalable cloud storage.

🖥️ Key Features & UI Walkthrough
The dashboard is structured to give users an immediate, clear snapshot of their financial health:

💳 Total Balance Card: Displays the user's aggregated wealth across all linked accounts, complete with a dynamic percentage indicator (📈 +2.4%) showing month-over-month growth trends.

📊 Icon-Enhanced Budget Status: Features visual progress bars paired with intuitive, colorful category icons:

🏠 Rent & Housing

🛒 Groceries

🍿 Entertainment
This graphic reinforcement allows users to instantly gauge their spending thresholds at a glance.

📅 Upcoming Bills Planner: A proactive financial calendar widget highlighting upcoming due dates with a quick-action ✔️ Settle button to prevent late fees and optimize monthly cash flow.

🧾 Recent Transactions Ledger: A dynamic, real-time data table that maps financial activity. The ledger uses corresponding visual context icons (like a 🛍️ grocery bag or a 🔌 power plug for utilities) for rapid categorization by date, type, and amount.

➕ Global Actions: A persistent floating action button (+ Add Transaction) and an intuitive sidebar menu make logging income or expenses seamless from any screen.

✨ Design & UX Highlights
🧠 Visual Recognition: The integration of tailored iconography drastically reduces cognitive load, allowing users to scan their spending habits in a split second.

🎨 Visual Hierarchy: Uses a balanced color palette—neutral, clean backgrounds paired with a vibrant neon green accent for positive metrics and primary action states.

🇵🇰 Multi-Currency Support: Features native localization, including localized currency displays (such as Pakistani Rupee - PKR) for targeted regional tracking.

⚡ Speed & Accessibility: Large metric callouts, interactive progress trackers, and direct graphic indicators ensure users can assess their financial status in under three seconds.
